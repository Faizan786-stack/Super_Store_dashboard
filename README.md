# Super_Store_Analysis_dashboard
## Project Overview

This Power BI project analyzes supermarket sales data to gain meaningful business insights and support better decision-making. The interactive dashboard provides a clear view of sales performance, customer behavior, and product trends using real-world data visualization techniques.

## Project Objective

The main goals of this project are to:

Analyze overall sales performance

Understand customer purchasing patterns

Identify top-selling products and categories

Track revenue trends over time

Create an interactive and user-friendly dashboard

## Key Insights & Features

 Total sales, revenue, and profit overview

 Category-wise and product-wise sales analysis

 Customer segmentation and behavior insights

 Time-based analysis (daily, monthly, yearly trends)

 Interactive filters and slicers for better exploration

## Tools & Technologies

Power BI Desktop

Power Query for data cleaning and transformation

Data Visualization best practices

## Dashboard Highlights

Clean and professional layout

Easy-to-use interactive visuals

KPIs for quick performance tracking

Business-focused insights for decision making

## Outcome

This dashboard helps stakeholders quickly understand sales trends, identify growth opportunities, and make data-driven business decisions.
